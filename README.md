# Radiant Elegance Chatbot

## Project Description
A simple rule-based Python chatbot for the Instagram skincare business **"Radiant Elegance"**.  
The chatbot answers FAQs about products, pricing, delivery, payment, and location.  
It handles **5 main user intents** and includes **polite greetings and informative responses**.

---

## How It Works
- Built in **Python** using **if-else logic**.  
- The user types a question or greeting in the terminal.  
- The chatbot responds based on **keyword matching**.  
- Recognized intents:
  1. Greeting
  2. Location
  3. Products / Brands
  4. Price
  5. Delivery / Payment
---

1. Clone the repository or download the files.  
2. Ensure **Python** is installed on your computer.  
3. Open **terminal** or **VS Code** in the project folder.  
4. Run the chatbot:

```bash

5. Type questions or greetings to interact.
